# Endless Skill Runner (Prototype)

Tarayicida acarak calistir:
1. `index.html` dosyasini bir browserda ac.
2. `Z`, `X`, `C` ile skill sec.
3. Sol tik ile secili skilli sahaya koy.
4. `R` ile yeniden baslat.

## Skill Plani
- `Z` Jump Pad: cooldown `2.2s`, sahada omur `8s`, karakter ustune basinca tek kullanimlik guclu ziplama.
- `X` Dash Gate: cooldown `3.6s`, sahada omur `7s`, karakter gecince `0.58s` dash; dash sirasinda duvar kirilir.
- `C` Barrier: cooldown `4.8s`, sahada omur `6s`, mermilere karsi `3` vurusluk kalkan.

## Engel - Skill Eslesmesi
- Gap (bosluk) -> Z ile gecis
- Wall (duvar) -> X ile kirip gecis
- Turret + projectile -> C ile savunma

## Denge Notlari
- Oyun hizi zamanla artar.
- Spawn araliklari zorlukla kisalir.
- Hedef: dogru skilli dogru konuma koyarak sonsuz ilerlemek.
